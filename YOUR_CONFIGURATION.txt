========================================
HOW TO GET YOUR OPENAI API KEY
========================================

1. Go to: https://platform.openai.com/api-keys
2. Sign in or create an account
3. Click "Create new secret key"
4. Copy the key (it starts with "sk-")
5. Replace "YOUR_OPENAI_API_KEY_HERE" above with your actual key

IMPORTANT: 
- Keep your API key secure and never share it
- OpenAI API requires billing to be set up
- Check pricing at: https://openai.com/pricing


========================================
NEXT STEPS TO RUN THE APPLICATION
========================================

1. Create .env file with the configuration above
   (make sure to add your OpenAI API key)

2. Create PostgreSQL database:
   createdb interview_db

3. Install dependencies:
   pip install -r requirements.txt

4. Initialize database:
   python init_db.py

5. Start the application:
   python run.py

6. Access the application:
   http://localhost:5000
